# qt-
一个由qt制作的c++图形化小游戏，简化版本的植物大战僵尸
图源网络 仅用作大作业 非商业用途
--------------------------------------------
ver1.0
需要在qt5下运行 qt6会报一堆错
好多bug 但能玩
--------------------------------------------
运行方法：1.若有qt5——release中下载压缩包 解压后找到一个后缀为.pro的文件 运行 如果跳出报错 点确定 再点击后面的configure.... 之后可正常构建运行
2.若无qt5，需在windows环境下下载游戏（安装链接：），点击后缀为.exe的文件游玩
